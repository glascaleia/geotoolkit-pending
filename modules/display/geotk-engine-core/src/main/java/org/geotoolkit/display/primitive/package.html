Implementation of {@linkplain org.geotools.display.primitive.GraphicPrimitive2D graphic primitives} to be rendered into a {@linkplain org.geotools.display.canvas.BufferedCanvas2D canvas}. An explanation for this package is provided in the {@linkplain org.opengis.go.display OpenGIS® javadoc}. The remaining discussion on this page is specific to the Geotools implementation.

The {@link org.geotools.display.canvas} package contains a discussion about rendering coordinate system applicable to all {@link org.geotools.display.primitive.GraphicPrimitive2D} objects, compared to the AWT coordinate system.
