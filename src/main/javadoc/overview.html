Overview of Geotoolkit-Pending.org.

Pending is the incubator

Geotk-pending is an incubator, modules may enter or exit without any obligations (like using geotoolkit-logging or backward compatibility). The objective of this project is to host new module that should end in the official Geotk modules. Each module has different quality level of code, testing and documentation. Some modules have reached a good stability for several months now like :
- geotk-style
- geotk-go2-engine
- geotk-filter
- geotk-wms

The incubator offer plenty of interesting new features that you can use, just keep in mind that the classes you use may be changed, renamed, moved or in the worse case removed at any time.
