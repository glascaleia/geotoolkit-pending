This is the base directory for convenience scripts executing
Geotoolkit.org command line tools. The directories are:

bin: contains the executable scripts. Users can add this
     directory to their path if desired.

etc: contains configuration (logging, JDBC drivers). Users can
     modify those file for fitting a particular configuration.
     However the default setting should be suitable for most
     usages.

jar: a symbolic link to the directory containing every JAR files
     built or downloaded by Maven. Users can change this link to
     a more permanent place if desired.
